Notable BST Data Structures

Here I note some famous BST data structures and which performance improvments they hold as defined in the section before.


Splay Tree

In short, a tree that moves every accessed element to the root via "splaying".

  • Static Optimality Bound✔
  • Static Finger Bound✔
  • Unified Bound with overhead O(n log n)✘
  • Worst case access O(n)✘

Multi-Splay Tree

A variant of Tango Trees. Improves on Tango Tree with a better worst case query performance.

  • O(log log n)-competitive ✔
  • Static Optimality Bound ✔
  • Static Finger Bound ✔
  • Unified Bound with overhead O(n log n)✘
  • Worst case access O(log^2 n)✘

Layered Working Set Tree

This tree guarentees the working set bound at worst case.

  • Static Finger Bound ✔
  • Unified Bound with overhead O(log log n)✔
  • Worst case access O(log n)✔
